Similar to a “drop quote”, words are dropped into the boxes in order to create a plausible 5-word clue that defines the “star” named in the rightmost column. For example, “DEBUTED ON 1924 VAUDEVILLE CIRCUIT” for BOB HOPE 2. Note the symbols following the names - four dashes and a star, with the star in varying positions. There are five words to each clue; highlight the one corresponding to the star’s position 3. The stars in boxes (12 total) should hint at the 12 starred words. These are: CIRCUIT, CLASSIC, ROUGH, OFF, RIDER, SOUND, MEDITATION, ELEMENT, OPPOSITE, LEADER, POISON, INTEGRATED. Each group yields a two-letter answer, which works nicely with the 10-letter final answer enumeration.
